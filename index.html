How I think — questions I'm chasing

My notebook, unfiltered

Robot self-knowledge

How does a robot truly know a task is done — is tactile or depth sensing a must, or can we do better?

Most pipelines lack valid environment monitoring. I keep probing where perception is necessary versus where it's a crutch.

Error & recovery

How does error accumulate across a pipeline, mathematically — and how do robots auto-recover from a wrong pose?

Sensor resolution, motor offsets, drift. Could shared memory or KV-cache-like structures help agents in one body recover together?

Learning at inference time

I lean toward Sutton's view: give an RL agent a minimal toolset and let it learn from inference time, exploring freely.

Single teleop no longer scales the data wheel. Watching world models, action-conditioned models, and long-horizon compression closely.

Contrarian habit

One assumption to doubt, always: what are the hidden assumptions in modern LLM-driven robotics?

My rule for staying honest — "keep the trees in sight; don't only walk on the meadow."

Currently studying → CS336 · modern control & MPC · inference frameworks · formal math · decentralized agent identity (ANP, zyx.awiki.ai)
